## Hardware Requirements
1. Raspberry Pi 3/4 2GB+
2. 90G Servos (1-16pcs, depending on the quantity of automatic slots)
3. 16GB+ MicroSD card (for the RPi)
4. 7" Waveshare Touchscreen (Other touchscreens should work too)
